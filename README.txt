How to run the ChatBot & Image Generator AI Project

1. Extract the file and copy "chatbot" folder

3.Paste it inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/HTML)

4.Open PHPMyAdmin at (http://localhost/phpmyadmin)

5. Create a database with the name "chatbot_db"

6. Import "chatbot_db.sql" file(given inside the zip package in the SQL file folder)
